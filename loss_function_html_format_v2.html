Introduction

This blog post aims to explain the fundamental concepts behind the Loss Functions in supervised learning. Before plunging into this topic, it is worth briefly defining the supervised machine learning, its objective and applications.

Basically, in the supervised machine learning, given a sample of target–predictor pairs \((y, x)\), we want to find a way to predict or estimate target values given their corresponding predictors. This sentence includes all you need to know about machine learning, and can be neatly translated into the following machine learning terms:

  1. "given a sample of target–predictor" refers to the machine learning training set.

  2. "find a way to predict" refers to the machine learning model and algorithm we pick for prediction.

Mathematically speaking, given \((y_i, x_i)\), find function \(\hat{y}_i=h_{\theta}(x_i)\) to estimate \(y_i\) for \(i=1,2,\cdots,N\), and predict new target values corresponding to a new predictor, \(x_{new}\), i.e., \(y_{new}=h_{\theta}(x_{new})\). The choice of \(h_{\theta}(x)\) defines our machine learning model. It is common to set \(h_{\theta}(x)=g(\theta^T x)\) which leads to a special class of machine learning models, called linear models.

Machine Learning Model and Algorithm

In simple words, function \(g(\cdot)\) is our machine learning model. However, function \(g(\cdot)\) and parameter \(\theta\) are still unknown. The process of picking the right and plausible function \(g(\cdot)\) is a design decision and it varies depending on the problem. For example, if the target values are binary, a good choice for \(g(\cdot)\) is the sigmoid function. Choosing the right and best model requires knowledge, experience and creativity.

After selecting the right model, we still need to know \(\theta\). This part is my favorite, since it boils down to the old fashion mathematical optimization. Mathematically, there are infinite choices for \(\theta\), but we want to find the best one. To that aim, we need to have some sort of measure to compare the different values of \(\theta\). One might say, I want a measure to minimize the distance of estimate vector,

\[\hat{Y}=\left[\begin{array}{c} h_{\theta}(x_1)\\ h_{\theta}(x_2)\\ \vdots\\ h_{\theta}(x_N)\\ \end{array} \right]\]

to the actual observed vector

\[Y=\left[\begin{array}{c} y_1\\ y_2\\ \vdots\\ y_N\\ \end{array} \right].\]

Let's say, \(D(Y,\hat{Y})\) is the distance between \(Y\) and \(\hat{Y}\). Then, we want to find \(\theta\) that minimizes \(D(Y,\hat{Y})\). Note that in the metric space, the mapping \(D(\cdot)\) should have certain properties, first, \(D(Y,Z)\geq0\), second, \(D(Y,Z)=0\) if and only if \(Y=Z\), and third, it must satisfy the triangular inequality. The common choices for \(D(\cdot)\) are the \(L_0\)-norm, the \(L_1\)-norm (Manhattan distance) and the \(L_2\)-norm (Euclidean distance). Note that \(D(\cdot)\) is called loss function in machine learning theory. Note that this function is also called cost function in statistics and optimization. Before jumping to the next step, which is the optimization algorithm that minimizes \(D(Y,\hat{Y})\) numerically, let’s discuss the loss function \(D(\cdot)\) in details

Common Loss Functions

So far, we assumed that we've picked our model right, i.e., we know the function \(h_{\theta}(x)\), we have our loss function \(D(Y,\hat{Y})\) in hand and we want to minimize it to get the best \(\theta\). But, what happens if we choose another loss function? Let's take a close look at common loss functions and their properties here:

  1. \(L_0\)-norm or All or nothing loss : \[D(Y, \hat{Y}) = \sum_{i=1}^{N}{I(y_i - h_{\theta}(x_i))}\] where \[I(y_i - h_{\theta}(x_i)) = \left\{\begin{array}{cc} 1 & |y_i - h_{\theta}(x_i)|\neq 0\\ 0 & |y_i - h_{\theta}(x_i)|= 0 \end{array}\right.\]

    • This loss function is not convex and not continuous
    • Computationally hard to minimize
    • Put a same weight on all error values

  2. \(L_1\)-norm or Absolute value loss: \[D(Y, \hat{Y}) = \sum_{i=1}^{N}{|y_i - h_{\theta}(x_i)|}\]

    • This loss function is convex, continuous and non smooth
    • Not sensitive to outliers
    • Solved as a linear program

  3. \(L_2\)-norm or Quadratic loss: \[D(Y, \hat{Y}) = \sum_{i=1}^{N}{(y_i - h_{\theta}(x_i))^2}\]

    • This loss function is convex, continuous and smooth
    • Sensitive to outliers
    • Solved by nonlinear optimization algorithms like gradient descent

  4. Huber loss with parameter \(\delta\): \[D(Y, \hat{Y}) = \sum_{i=1}^{N}{H_{\delta}(y_i - h_{\theta}(x_i))},\] where \[H_{\delta}(y_i - h_{\theta}(x_i)) = \left\{\begin{array}{cc} (y_i - h_{\theta}(x_i))^2 & |y_i - h_{\theta}(x_i)|\leq \delta\\ 2\delta|y_i - h_{\theta}(x_i)|-\delta^2 & |y_i - h_{\theta}(x_i)|\geq \delta \end{array}\right.\]

    • This loss function is convex, continuous and non smooth
    • Sensitive to errors less than \(\delta\)
    • Solved by non-smooth optimization algorithms such as sub-gradient methods

  5. Inverse huber loss with parameter \(\delta\): \[D(Y, \hat{Y}) = \sum_{i=1}^{N}{H^{-1}_{\delta}(y_i - h_{\theta}(x_i))},\] where \[H^{-1}_{\delta}(y_i - h_{\theta}(x_i)) = \left\{\begin{array}{cc} |y_i - h_{\theta}(x_i)| & |y_i - h_{\theta}(x_i)|\leq \delta\\ \frac{2(y_i - h_{\theta}(x_i))^2-\delta^2}{\delta} & |y_i - h_{\theta}(x_i)|\geq \delta \end{array}\right.\]

    • This loss function is convex, continuous and non smooth
    • Sensitive to errors greater than \(\delta\)
    • Solved by non-smooth optimization algorithms such as sub-gradient methods

Among these loss functions, quadratic loss or squared loss is more common due to its numerical and analytical properties. However, it is not robust for long tailed distributed data. Generally, as it is depicted in the following plot of one dimensional loss functions, x-axis is the error and y-axis is the penalty associated with that error. The absolute value loss tends to penalize small errors more than quadratic loss. On the other hand, the quardatic loss function penalizes large errors more. With the huber and inverse huber loss functions, you can control this penalty using the huber parameter.

Now, let’s get back to our main concern, finding \(\theta\). Well, we setup our problem right and get to this point where we want to solve the following problem: \[\min_{\theta} D(Y, \hat{Y})\] Depending on the choice of a convex loss function, you either have a smooth or non-smooth optimization problem to solve. If the optimization problem is smooth, you can use gradient descent algorithm to solve it. In the next blog, we will discuss these numerical optimization algorithms.

Conclusion

In this technical post, we discussed different loss functions, properties and applications. This post would help you to decide which loss function is better for your machine learning model considering the real world problem and computational complexities.
